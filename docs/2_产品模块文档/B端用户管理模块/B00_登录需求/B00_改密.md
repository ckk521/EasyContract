# B00_改密

## Story

**作为B端admin，首次登录时我想要强制修改密码，以确保账户安全**

**验收标准**：
- 首次登录（first_login=true）强制跳转到修改密码页面
- 修改密码后才能进入B端后台
- 新密码至少8位，包含大小写字母和数字
- 新密码不能与当前密码相同

---

### 页面元素说明

| 元素 | 说明 | 必填 |
|------|------|------|
| 当前密码输入框 | 验证身份 | 是 |
| 新密码输入框 | 设置新密码 | 是 |
| 确认密码输入框 | 确认新密码 | 是 |
| 密码规则提示 | 显示密码要求 | - |
| 确认修改按钮 | 提交修改 | - |

### 交互说明

- 首次登录（first_login=true）强制跳转到此页面
- 修改密码后才能进入B端后台
- 登录成功后跳转到 /b-side

---

## 路由设计

```
/b-change-password → 首次登录修改密码页面
/b-side             → B端主后台（需登录）
```

### 路由守卫

- 访问 /b-side 需先检查登录状态
- 已登录但 first_login=true 强制跳转 /b-change-password
- 已登录且 first_login=false 才能进入B端后台
