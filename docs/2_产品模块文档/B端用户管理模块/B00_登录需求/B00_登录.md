# B00_登录

## Story

**作为B端admin，我想要登录后台，以便管理系统**

**验收标准**：
- 预设admin账号密码登录
- 登录后跳转B端管理后台

---

### 页面元素说明

| 元素 | 说明 | 必填 |
|------|------|------|
| 用户名输入框 | B端员工用户名 | 是 |
| 密码输入框 | 支持显示/隐藏切换 | 是 |
| 登录按钮 | 主按钮，深色背景 | - |
| 忘记密码链接 | 联系管理员重置 | - |

### 交互说明

- 登录页面左侧展示系统Logo和介绍
- 右侧为登录表单
- 登录成功后跳转到 /b-side 或 /b-change-password（首次登录）

---

## 路由设计

```
/b-login          → B端登录页面
/b-side           → B端主后台（需登录）
```

### 路由守卫

- 访问 /b-side 需先检查登录状态
- 未登录跳转 /b-login
- 已登录但 first_login=true 跳转 /b-change-password
- 已登录且 first_login=false 进入B端后台
