# B01 - 登录与权限需求

## Story

**作为B端admin，我想要登录后台，以便管理系统**

**验收标准**：
- 预设账号密码登录
- 首次登录强制修改密码

**作为B端admin，我想要管理C端用户，以便控制访问权限**

**验收标准**：
- 启用/禁用/删除用户
- 查看用户信息
