# C01 - C端登录注册需求

## Story

**作为C端用户，我想要注册账号，以便使用智能体服务**

**验收标准**：
- 手机号/邮箱 + 密码注册
- 密码至少6位
- 用户名唯一

**作为C端用户，我想要登录系统，以便开始使用**

**验收标准**：
- 账号密码登录
- Token有效期7天
- 登录后跳转C端对话页面

### 页面元素说明

| 元素 | 说明 | 必填 |
|------|------|------|
| 账号输入框 | 支持手机号或邮箱 | 是 |
| 密码输入框 | 支持显示/隐藏切换 | 是 |
| 记住登录状态 | 复选框，默认不勾选 | 否 |
| 登录按钮 | 主按钮，蓝色背景 | - |
| 注册链接 | 跳转到注册页面 | - |
| 忘记密码链接 | 跳转到找回密码页面 | - |

### 交互说明

- 输入框获焦时，边框变为蓝色
- 错误时，边框变为红色，显示错误提示
- 登录/注册按钮点击后显示loading状态
- 登录成功后跳转到 /c-side
- 注册成功后跳转到登录页
