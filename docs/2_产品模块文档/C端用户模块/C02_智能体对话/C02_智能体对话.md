# C02 - 智能体对话需求

## Story

**作为C端用户，我想要与智能体对话，以便获得法律咨询**

**验收标准**：
- 多轮对话，上下文理解
- 响应≤3秒
- SSE流式响应
- 识别用户意图并执行对应操作

**作为C端用户，我想要智能体委婉拒绝非法律问题，以便不浪费时间**

**验收标准**：
- 使用B端配置的拒绝话术
- 引导回归法律范畴
