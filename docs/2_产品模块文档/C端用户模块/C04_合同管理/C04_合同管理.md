# C04 - 合同管理需求

## Story

**作为B端用户，我想要定义合同类型，以便规范合同分类**

**验收标准**：
- 创建/编辑/删除合同类型
- 合同类型包含：类型编码、名称、描述
- 支持启用/禁用

**作为B端用户，我想要上传合同模板，以便C端用户使用模板生成合同**

**验收标准**：
- 选择合同类型后上传模板
- 模板支持 Markdown + 占位符格式
- 上传时自动识别占位符并映射字段
- 支持手动补充字段定义

**作为B端用户，我想要管理合同字段，以便自定义合同内容**

**验收标准**：
- 从组件库选择字段或自定义新字段
- 字段支持分组管理
- 字段支持验证规则（必填、长度、格式等）
- 字段支持条件显示（如下拉选项联动）
- 支持设置字段在正文中的占位符位置

**作为B端用户，我想要查询和管理合同列表，以便掌握合同状态**

**验收标准**：
- 支持按类型、状态、日期等筛选
- 支持关键字搜索
- 支持查看合同详情
- 支持修改合同内容（C端提交前）

---

**作为C端用户，我想要管理我的合同，以便查看不同合同的状态**

**验收标准**：
- 合同列表、状态显示
- 切换会话
- 草稿继续填写

**作为C端用户，我想要生成租赁合同，以便快速获得合同文档**

**验收标准**：
- 逐个提问引导填写字段
- 必填项校验
- AI润色可选
- 导出PDF

---
