# C04 - 合同管理需求

## Story

**作为C端用户，我想要管理我的合同，以便查看不同合同的状态**

**验收标准**：
- 合同列表、状态显示
- 切换会话
- 草稿继续填写

**作为C端用户，我想要生成租赁合同，以便快速获得合同文档**

**验收标准**：
- 逐个提问引导填写字段
- 必填项校验
- AI润色可选
- 导出PDF
