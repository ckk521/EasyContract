# R04 - 预检测需求

## Story

**作为AI审核系统，我想要进行预检测，以便快速识别确定性问题**

**验收标准**：
- 检测无效字段值（身份证格式、电话格式等）
- 检测缺失字段（空白占位符）
- 生成风险项，不依赖AI判断
