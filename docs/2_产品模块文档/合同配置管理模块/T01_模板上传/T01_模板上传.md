# T01 - 模板上传需求

## Story

**作为B端用户，我想要上传Word模板，以便智能体可以使用这些模板生成合同**

**验收标准**：
- 支持.docx格式
- 自动校验格式要求
- 变量标记【】配对
- 下划线长度≥5个

**作为B端用户，我想要预览模板解析结果，以便确认变量识别正确**

**验收标准**：
- 保留Word格式
- 可点击的标记点
- 已配置字段显示字段名
- 未配置字段显示提示
