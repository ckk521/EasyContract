# T02 - 字段配置需求

## Story

**作为B端用户，我想要配置字段映射，以便将模板变量绑定到字段库**

**验收标准**：
- 变量必须映射到字段
- 留白可选映射或不填写
- 支持新建字段并保存到字段库
- 实时保存配置
