# T03 - 合同类型管理需求

## Story

**作为B端admin，我想要管理合同类型，以便支持更多合同类型**

**验收标准**：
- 查看/新增/编辑/启用/禁用合同类型
- 每种类型有唯一编码和名称

**作为B端admin，我想要管理字段库，以便复用字段定义**

**验收标准**：
- 查看/新增/编辑/删除字段
- 字段按类型分组
- 检查字段是否被模板使用
